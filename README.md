# AI Systems Weekly

一个持续公开记录 AI 系统、Agent、自动化工作流、工具、论文、新闻与实战观察的内容仓库。

这里更像一份长期演进的研究与判断档案，而不是代码项目主页。脚本只负责降低维护成本，核心始终是内容本身。

- [收件箱 / 选题池](./docs/inbox/README.md)
- [内容模板](./docs/templates/)

## 仓库包含什么

- AI 系统与 Agent 的结构化观察
- 自动化工作流与工具链记录
- 论文、产品、行业新闻的筛选与判断
- 个人方法论、长期主题与阶段性结论

## 目录说明

```text
.
├─ docs/
│  ├─ daily/           # 日更记录
│  ├─ weekly/          # 周刊整理
│  ├─ inbox/           # 选题、线索、待整理素材
│  └─ templates/       # 日更 / 周刊 / 论文 / 评测 / RAG 模板
├─ scripts/            # 轻量辅助脚本
├─ push.bat            # Windows 一键提交
└─ push.ps1            # Windows PowerShell 提交脚本
```
